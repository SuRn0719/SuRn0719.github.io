Hexo 常用的部分快捷键

发表于 2019-01-27 | 分类于 Hexo
字数统计: 496 | 阅读时长 ≈ 1

hexo s

启动本地服务器，用于预览主题和内容，默认地址为：http://localhost:4000/。
预览的同时可以修改文章内容、代码或者是主题的配置文件，保存直接刷新就可以了。
如果是修改Hexo默认的配置文件（ _config.yml ）的话，需要重启才生效。

hexo n

新建一篇标题为文章名字的文章，如果有空格，需要打引号。
标题可以在“文章名字.md”文件里面修改。
文章可以拥有以下属性：
1.layout：布局，默认post或page。
2.title：文章的标题。
3.date：创建日期，文件的创建日期。
4.update：修改日期或叫更新日期，文件的修改日期或更新日期。
5.comments：是否开启文章评论，true或false，true打开，false关闭。
6.tags：标签，设置文章的标签。
7.categorys：分类，设置文件的分类
8.permalink：url中的名字，文件名。

hexo d

自动生成网站静态文件，并部署到设定的仓库。

hexo clean

清除缓存文件 db.json 和已生成的静态文件 public。
当网站显示异常时，可以试试这条命令清除以下缓存文件。

hexo g

生成网站静态文件到默认设置的 public 文件夹。
便于查看网站生成的静态文件或者手动部署网站；
如果使用自动部署，不需要先执行该命令。

hexo n page

新建一个标题为 aboutme 的页面，默认链接地址为 主页地址/aboutme/,
标题可以为中文，但一般习惯用英文；
页面标题和文章一样可以随意修改；
页面不会出现在首页文章列表和归档中，也不支持设置分类和标签。

这些是目前我用到的，部分的注释也是来自网上的，自己只稍微修改了一下，方便自己去理解。
